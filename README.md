# VoxiProxyTun

Клиент для подключения к прокси-серверам по подписке.

## Скачать

[Последний релиз](https://github.com/Sweerx/voxiproxy-app/releases/latest)

## Установка (Windows)

1. Скачайте `.exe` из Releases
2. Запустите установщик
3. Вставьте ссылку подписки
4. Выберите сервер и подключитесь
